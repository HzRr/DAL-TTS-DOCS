DAL TTS API

基于VITS的约战多角色语音合成接口

本功能仅供学习和交流使用，所造成任何后果皆由使用者本人承担

# API 用法

# 1.网页访问

**使用前先看文档! **
网页链接(有文本长度限制)(暂时关闭): 点击访问open in new window
新增长文本语音合成(推荐): 点击访问open in new window
使用方法:

1.选择需要生成语音的角色
2.选择模式:

模式状态描述
日语可用输入日语，直接生成语音
中文可用输入中文，自动翻译为日语后生成语音
拼音不可用输入中文，将中文空耳成相应日语后生成语音

3.点击生成按钮，等待语音生成
PS: 因为模型在服务器上跑，用的CPU推导，因此速度比不上GPU推导，并且文本长度也会加以限制(22个字符，不包括空格)
语气停顿可以通过。，.,！等符号来实现(计算在22个字符内)，语气现在只能通过在文本中加入↑或↓来调节
